= RPackager {<img src="https://travis-ci.org/narojigautam/r_packager.svg?branch=master" />}[https://travis-ci.org/narojigautam/r_packager]

Hosted on Heroku at http://r-packager.herokuapp.com
Currently we are indexing packages from CRAN servers only.
